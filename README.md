# online-banking-system
Bankμ is a simulation of an online banking system designed to demonstrate the core functionalities of digital banking. This project allows users to perform banking operations such as creating accounts, logging in, checking balances, transferring funds, and viewing transaction history.

Note: This is a simulation for educational purposes only. No real money transactions are performed.

# Project Structure
- [index.html:](index.html) Landing page of the website
- [auth.html:](bankμ/auth.html) Login and registration forms
- [dashboard.html:](bankμ/dashboard.html) User dashboard after login
- [style.css:](bankμ/style.css) All CSS styles for the website
- [auth.js:](bankμ/auth.js) JavaScript for login and registration functionality
- [dashboard.js:](bankμ/dashboard.js) JavaScript for dashboard functionalities
- [banku_db.sql:](bankμ/banku_db.sql) SQL database template

# How to Run
1. Clone or download the repository.
2. Place the project folder inside your web server’s root directory.
3. Set up the database using your preferred SQL database system.
4. Start your web server and database services.
5. Open a web browser and navigate to the project directory via localhost.
